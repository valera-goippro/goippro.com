Cómo Empezar Earning

15–20 minutes to set up. First payout in 30 days.

What You Need

Equipment

GoIP gateway — the main device. Accepts 1–32 SIM cards, handles SMS and voice traffic. Or an Android phone (8.0+) as SMS-only alternative.

How to choose a gateway →

SIM Cards

Any carrier, any country. We recommend a plan with a large bundle of minutes and SMS. The app uses 1–5 GB of data per port per month.

Crypto Wallet & Telegram

USDT (TRC-20) or USDC (ERC-20) for payouts. Trust Wallet, Binance, OKX all work. Telegram is required — device status notifications come through it.

Five Steps

Registration

Go to goippro.com — create an account: email, password, country. Confirm email. Takes 2 minutes.

Connect Telegram

Dashboard → Notifications → Connect Telegram → Start the bot → enter code. Required: without Telegram you won’t know about device issues in time.

Connect Equipment

For GoIP gateway — via API in your dashboard. For Android phone — download APK, install the app, grant all permissions. Important: disable battery optimization for GoIPPro in phone settings.

Add SIM Cards

The system will automatically register SIM cards. Status changes to "Validation" — 10 business days of stability assessment. Then "Active" — and earnings begin.

Set Up Wallet

Profile → Wallet → paste your USDT or USDC address. Double-check the address — payouts are non-reversible. First payout arrives at the start of the next month.
