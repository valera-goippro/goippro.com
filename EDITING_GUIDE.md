# GoIPpro.com — Руководство по редактированию

## Что МОЖНО редактировать

### Тема WordPress
- `wp-content/themes/developer-flavor-developer-flavor-developer/` — основная тема
- Шаблоны страниц, стили CSS, JavaScript
- Изображения в теме

### Контент и медиа
- `wp-content/uploads/` — загруженные изображения и файлы
- Статические HTML-страницы в корне (если есть)

### Плагины (осторожно)
- `wp-content/plugins/` — можно обновлять файлы плагинов, но осторожно

## Что НЕЛЬЗЯ редактировать

- `wp-includes/` — ядро WordPress, не трогать
- `wp-admin/` — админка WordPress (не в репо)
- `wp-config.php` — конфигурация БД (не в репо)
- `.htaccess` — настройки сервера (не в репо)
- `wp-login.php`, `wp-cron.php` и другие корневые PHP-файлы WordPress

## Как вносить изменения

1. Найти нужный файл в репо
2. Внести правки (текст, CSS, изображения)
3. Сделать commit с понятным описанием
4. Push в ветку `main`
5. GitHub Actions автоматически загрузит изменения на сайт (~1-2 мин)

## Проверка деплоя

- GitHub → Actions → последний workflow run
- Зелёная галочка = деплой успешен
- Красный крестик = ошибка, проверить логи
