== Description ==

Modern WordPress theme with React-powered interface for Soft24 platform.

== Installation ==

1. Upload the theme folder to /wp-content/themes/
2. Activate the theme through the 'Themes' menu in WordPress
3. Done! The theme is ready to use.

== Features ==

- Modern React-based interface
- Responsive design
- Fast loading
- Clean and minimal
